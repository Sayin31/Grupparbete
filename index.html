HTML 4

  • 4:e HTML versionen som funkar i de flesta ´browsers´, den är dock mer komplex och ser inte lika bra ut.

  • Den har längre koder som gör svårare att kompilera enkelt kod. ex.(<!DOCTYPE> HTML PUBLIC ”-//W3C//DTD HTML 4.01/EN” ”http://www.w3.org/TR/html4/strict.dtd” Här måste man skriva från vilken source koden kommer ifrån>.

  • Html4 har inte lika bra felhantering och kräver mer uppmärksamhet från användaren att kompilerat allt på korrekt sätt.

  • I Html4 så finns det inte taggar som canvas, video eller audio; Men den innehåller mer taggar än Html5 som gör att kompilationen blir onödigt längre.

  • I Html4 så stöds multimedian av utomstående tillägg så som Silverlight eller Flash.

HTML 5

  • Senaste HTML versionen med nya tags samt med möjligheter till bättre utseende.

  • Allt kod i HTML blev simplifierad för användaren, då man behöver inte skriva lika mycket kod. (ex.< !DOCTYPE> Här räcker det med att skriva denna paragraf).

  • Html5 har ett bra felhantering system som hjälper användaren att se snabbt vad är fel kompilerad, utöver det så kan systemet fixa detta fel automatisk.

  • I Html5 har de tagit bort eller simplifierad lera taggar och addad nya som inte har funnits tidigare så som canvas, video och audio.

  • Html5 har tillgång till inbyggt multimedia.
